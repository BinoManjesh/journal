 1.Accept the size of the matrix in n
 2.Accept the elements of the matrix in mat
 3.If n == 1, print mat[0][0] and exit
 4.Let sum = 0
 5.For i from 0 to n - 1:
 6.    If i != 0 or i != n - 1
 6.        Add mat[0][i] to sum
 7.        Add mat[n - 1][i] to sum
 8.    Add mat[i][0] to sum
 9.    Add mat[i][n - 1] to sum
10.    If i != 0 and i != n - 1:
11.        Add mat[i][i] to sum
12.        If n - i - 1 != i
13.            Add mat[i][n - i - 1]
14.Print sum
